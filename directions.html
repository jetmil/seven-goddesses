О Наборах Свечей

Каждое направление жизни требует не одной свечи, а набора — комплекса из нескольких свечей разной формы, цвета и назначения. Набор работает как единый организм: каждая свеча выполняет свою функцию, вместе они создают полноценное воздействие.

Типичный набор включает:

  • Главная свеча — несёт основную задачу (скрутка, коса или формовая)
  • Усилитель — добавляет энергию (обычно скрутка или спираль)
  • Якорь — закрепляет результат (формовая свеча: монета, куб, звезда и т.д.)
  • Расцеп — для очищения и разрыва связей (плетение из 4 свечей в виде цепи, только когда нужно что-то расцепить)
  • Громничная — защищает работу (по желанию, но рекомендуется)
Направление I

Личность и Начинания

Самоопределение, запуск новых проектов, первый шаг, инициатива, решительность, преодоление инерции, лидерство, самопрезентация

Ведущая: Хефаэлла | Помощницы: Амаресса, Люминара

Набор Свечей

Состав набора: 4 свечи

1. Главная — Скрутка Первого Шага

  • Тип: Тугая скрутка из 2 свечей
  • Цвета: красная + оранжевая
  • Направление: по часовой стрелке
  • Функция: прямой импульс действия

2. Усилитель — Спираль Жара

  • Тип: Одиночная свеча, обмотанная красной нитью по спирали
  • Цвет: белая свеча, красная нить
  • Функция: разгоняет энергию, убирает страх

3. Якорь — Формовой Камень

  • Тип: Формовая свеча в виде камня или куба
  • Цвет: коричневый или терракотовый
  • Функция: заземление, чтобы начатое не растворилось
  • Альтернативные формы: пирамида, гриб (рост из земли), жёлудь
  • Плетёный аналог: тугая короткая скрутка из 3 коричневых свечей (символ твёрдости)

4. Громничная — Защита Начала

  • Тип: Классическая громничная (тонкая белая, освящённая)
  • Цвет: белый или натуральный воск
  • Функция: защита проекта от сглаза и зависти

Геометрия Алтаря

★ Громничная (север) ◐ Спираль ◑ Камень (запад) (восток) ⊗ Скрутка (центр) ▼ Вход практика (юг)

Порядок зажжения:

  1. Громничная (защита пространства)
  2. Камень-якорь (создание основы)
  3. Спираль (разогрев энергии)
  4. Главная скрутка (запуск действия)

Травяной Состав

  • Для Скрутки: крапива + имбирь
  • Для Спирали: розмарин + корица
  • Для Камня: корень лопуха + дубовая кора
  • Для Громничной: травы не добавляются (сохраняет чистоту)

Благовоние

Ладан + корица + щепотка чёрного перца

Метод: Три круга по часовой вокруг алтаря перед зажжением.

Призыв

«Хефаэлла, Госпожа Горна, Мать Действия!
Ты, что бьёшь молотом первой,
Ты, что не знаешь слова «потом» —
Благослови мои руки на плетение.

Амаресса, Хранительница Свечей Судьбы,
Дай место моей работе на высоких полках.

Люминара, Несущая Свет,
Влей уверенность в каждое движение.

Четыре свечи — четыре опоры начала.
Скрутка — действие.
Спираль — разгон.
Камень — основа.
Громничная — защита.

Да будет так.»

Заговоры При Плетении

При скрутке (Главная):
«Красное с оранжевым вью,
Первый шаг — твёрдый.
Второй — решительный.
Третий — победный.
Скручиваю силу — иду.»

При обмотке спирали (Усилитель):
«Витком к витку,
Страх — в пепел,
Сомнение — в дым,

При формовке камня (Якорь):
«Камень — в основу,
Начатое — закреплено,
Проект — укоренён,
Не сдует, не унесёт.»

Заговор При Зажжении

  • Громничная: «Защита — первой. Щит — поставлен.»
  • Камень: «Основа — заложена. Фундамент — крепок.»
  • Спираль: «Жар — поднимается. Страх — уходит.»
  • Скрутка: «Действие — началось. Первый шаг — сделан. Пути — открыты.»

Пасы в Воздухе

  • Над набором: Треугольник вершиной вверх (огонь, действие)
  • Направление движения: снизу вверх, три раза
  • В конце: Резкий толчок ладонью вперёд (отправка намерения)

Время Проведения

  • Луна: Растущая (от новолуния до полнолуния)
  • День: Вторник (день Хефаэллы и Марса)
  • Час: Первый час после рассвета (начало нового)

Длительность Горения

  • Громничная: весь ритуал + догорает после
  • Остальные: одновременно, примерно 1-2 часа
  • Важно: Скрутка должна догореть полностью

Знаки Завершения

  • Пламя скрутки вспыхнуло особенно ярко перед угасанием
  • Воск громничной образовал форму стрелы или копья
  • Ощущение тепла и решимости в груди
  • Спонтанное желание встать и действовать
Направление II

Материальные Ресурсы

Деньги, имущество, заработок, ценности, накопления, достаток, материальная безопасность

Ведущая: Нурина | Помощницы: Эстиора, Хефаэлла

Набор Свечей

Состав набора: 5 свечей

1. Главная — Коса Изобилия

  • Тип: Коса из 3 свечей
  • Цвета: золотая/жёлтая + зелёная + белая
  • Направление: плетение от себя (отдача, чтобы получить)
  • Функция: открытие каналов получения

2. Усилитель — Скрутка Притяжения

  • Тип: Скрутка из 2 свечей
  • Цвета: зелёная + золотая
  • Направление: к себе (по часовой)
  • Функция: магнит для денег

3. Якорь — Формовая Монета

  • Тип: Круглая формовая свеча (в форме монеты или круга)
  • Цвет: золотой/жёлтый
  • Функция: накопление, сохранение
  • Альтернативные формы: слиток, подкова, сундук, куб (стабильность капитала)
  • Плетёный аналог: кольцо из 2 золотых свечей (свечи согнуть в круг и соединить концы)

4. Поток — Двойная Тонкая Скрутка

  • Тип: Две тонкие свечи, мягко скрученные вместе
  • Цвета: зелёная + золотая
  • Функция: постоянный приток + постоянный расход (здоровое движение денег)

5. Громничная — Защита Достатка

  • Тип: Классическая громничная
  • Цвет: белый
  • Функция: защита от воров, завистников, финансовых потерь

Геометрия Алтаря

★ Громничная (север) ◐ Двойная тонкая ◑ Скрутка притяжения (северо-запад) (северо-восток) ◎ Формовая Монета (центр) ⊗ Коса (юг, ближе к практику) ▼ Вход практика

Порядок зажжения:

  1. Громничная
  2. Формовая монета (центр накопления)
  3. Двойная тонкая (запуск потоков)
  4. Скрутка (притяжение)
  5. Коса (изобилие)

Травяной Состав

  • Для Косы: базилик + мята + корица
  • Для Скрутки притяжения: гвоздика + имбирь
  • Для Формовой Монеты: шафран (или куркума) + лавровый лист
  • Для Двойной тонкой: пажитник + кориандр

Благовоние

Мирра + сандал + гвоздика

Метод: Пять кругов по часовой стрелке. Дымом нарисовать знак бесконечности (∞) над алтарём.

Призыв

«Нурина, Госпожа Золотых Полей!

Заговор При Зажжении

  • Громничная: «Щит достатка — поставлен. Воры и завистники — мимо.»
  • Формовая: «Центр накопления — активен. Сохраняю и множу.»
  • Двойная тонкая: «Потоки — открыты. Деньги текут.»
  • Скрутка: «Магнит — включён. Притягиваю.»
  • Коса: «Изобилие — в мою жизнь. Закрома наполняются.»

Пасы в Воздухе

  • Над набором: Горизонтальная восьмёрка (∞) — бесконечность потока
  • Направление: плавно, три раза
  • В конце: Руки сложить в чашу и «вылить» на монету

Время Проведения

  • Луна: Растущая (от первой четверти к полнолунию)
  • День: Четверг (день Юпитера, расширения) или Пятница (день Венеры, ценностей)
  • Час: Полдень (максимум солнечной энергии)

Знаки Завершения

  • Пламя косы ровное, высокое, золотистое
  • Воск формовой монеты сохранил круглую форму
  • Запах мёда или хлеба во время горения
  • Спонтанные идеи о заработке во время ритуала
Направление III

Связи и Познание

Общение, обучение, информация, ближние контакты, переговоры, документы, короткие поездки, соседи, братья/сёстры

Ведущая: Софиэль | Помощницы: Люминара, Нурина

Набор Свечей

Состав набора: 4 свечи

1. Главная — Двойная Коса Связей

  • Тип: Две косы, сплетённые вместе
  • Цвета: голубые + белые + жёлтые
  • Функция: создание и укрепление связей

2. Усилитель — Скрутка Красноречия

  • Тип: Скрутка из 2 тонких свечей
  • Цвета: голубая + оранжевая
  • Функция: дар речи, убедительность

3. Якорь — Формовая Книга

  • Тип: Плоская формовая свеча (прямоугольник)
  • Цвет: голубой или белый
  • Функция: фиксация знаний
  • Альтернативные формы: свиток, перо, сова (мудрость), куб (структура)
  • Плетёный аналог: плоская коса из 3 голубых свечей, прижатая грузом при застывании

4. Громничная — Защита Слова

  • Тип: Классическая громничная
  • Функция: защита от клеветы, непонимания

★ Громничная (север) ◐ Скрутка ◑ Книга (запад) (восток) ⊗ Двойная коса (центр) ▼ Вход практика

Направление IV

Корни и Защита

Дом, семья, род, безопасность, недвижимость, родители, корни, основа существования

Ведущая: Эстиора | Помощницы: Моригена, Нурина

Набор Свечей

Состав набора: 5 свечей

1. Главная — Коса Рода

  • Тип: Толстая коса из 3 свечей
  • Цвета: коричневая + оранжевая + белая
  • Функция: укрепление родовой силы

2. Усилитель — Скрутка Тепла

  • Тип: Скрутка из 2 свечей
  • Цвета: оранжевая + жёлтая
  • Функция: тепло в доме, любовь в семье

3. Якорь — Формовой Дом

  • Тип: Формовая свеча в виде домика или куба
  • Цвет: коричневый или терракотовый
  • Функция: защита жилища
  • Альтернативные формы: очаг, печь, ключ (от дома), гнездо, дерево с корнями
  • Плетёный аналог: коса из 4 коричневых свечей, уложенная квадратом (символ стен)

4. Коса Поколений

  • Тип: Коса из 3 свечей (связывает три времени)
  • Цвета: чёрная (предки) + коричневая (я) + белая (потомки)
  • Функция: связь поколений

5. Громничная — Страж Порога

  • Тип: Усиленная громничная (толще обычной)
  • Функция: защита дома от зла

Геометрия Алтаря

★ Громничная (у порога/на север) ◐ Коса поколений ◑ Скрутка (запад) (восток) ◎ Дом (центр) ⊗ Коса рода (юг) ▼ Практик

Порядок зажжения:

  1. Громничная (у порога)
  2. Формовой дом (центр защиты)
  3. Коса поколений (связь с родом)
  4. Скрутка тепла (любовь)
  5. Коса рода (сила)

Травяной Состав

  • Для Косы Рода: чабрец + мать-и-мачеха + зверобой
  • Для Скрутки: ромашка + мелисса
  • Для Дома: полынь (защита) + можжевельник
  • Для Косы Поколений: корень валерианы + сушёный дуб

Благовоние

Можжевельник + полынь + капля мёда на угли

Метод: Обойти весь дом с дымом. Вернуться к алтарю, сделать три круга.
